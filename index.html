1. Initialize a Repository

Use git init to start versioning your project.

2. Clone an Existing Repository

Copy a project from a remote source using git clone <url>.

3. Stage Your Changes

Add files to the staging area with git add <file>.

4. Commit Your Changes

Record changes to the repository with git commit -m "message".

5. Check Status

View current changes using git status.

6. Check Log

See commit history with git log.

7. Manage Branches

Create, list, or delete branches using git branch.

8. Switch Branches

Move between branches with git checkout <branch>.

9. Merge Branches

Incorporate changes from another branch using git merge <branch>.

10. Pull Latest Changes

Update your local repository with git pull.

11. Push Changes

Upload your local changes to a remote repository using git push.
